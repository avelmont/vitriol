# Guide

Welcome to the Vitriol documentation guide. Here you'll find comprehensive resources to understand and work with workflows.

## Overview

Vitriol is a workflow orchestration system that enables you to define, execute, and manage complex data processing pipelines. This guide will walk you through the core concepts and practical examples.
